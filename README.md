# 🎮 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.

## 🚀 Live Demo

https://github.com/jyoti252006-dotcom/games_JS.git

---

## 📌 Features

- 🪨 Rock, 📄 Paper, ✂️ Scissors gameplay
- 🤖 Random computer choice
- 🏆 Real-time score tracking
- 🎨 Dynamic win/lose/draw messages
- 🌈 Message color changes based on game result
- 💻 Clean and responsive UI

## 🎯 How to Play

1. Click on **Rock**, **Paper**, or **Scissors**.
2. The computer randomly selects its move.
3. The winner is decided based on the game rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
4. Scores are updated automatically.
